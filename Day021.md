## Dicts: Number of occurrences

The text is given in a single line. For each word of the text count the number of its occurrences before it.

```
Example input
one two one two three two four three

Example output
0 0 1 1 0 2 0 1
```

## source code
```
a = input().split()
b= {}
for word in a:
  if word not in b:
    b[word] = 0
  print(b[word], end=' ')
  b[word] += 1
```
